# Box Hunter

## How to edit this project in Unity and build it yourself

1. If it is not already installed, [download Unity Hub](https://unity.com/download) and open it.

2. Download this repository and [add it as an existing project from your disk](https://docs.unity3d.com/hub/manual/AddProject.html#add-an-existing-project-from-your-disk).

## Como editar este proyecto en Unity y construirlo

1. Si no está instalado, [descarga Unity Hub](https://unity.com/es/download) y ábralo.

2. Descarga este repositorio y [añádelo como un proyecto existente desde tu disco](https://docs.unity3d.com/hub/manual/AddProject.html#add-an-existing-project-from-your-disk).
